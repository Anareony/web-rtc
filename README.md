### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Anareony/web-rtc.git
   ```
2. Navigate to the project directory
   ```sh
   cd web-rtc
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the project
   ```sh
   npm run dev
   ```
5. The project will run on
   ```sh
   http://localhost:5173
   ```
